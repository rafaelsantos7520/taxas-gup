# Taxas GUP

O **Taxas GUP** é um projeto gratuito voltado para a transparência na cobrança de entregas feitas por motoboys autônomos.

## Propósito do projeto

Este projeto foi criado para:

- exibir de forma simples e rápida as **taxas de entrega por região e cidade**;
- apoiar a **Associação de Motoboys Autônomos Independentes** na padronização e comunicação dos valores;
- facilitar a visualização dos preços para clientes, promovendo uma negociação clara;
- incentivar um **preço justo** para quem entrega e para quem contrata o serviço.

## Como funciona

A aplicação permite selecionar origem e destino para visualizar valores sugeridos de frete.

> Atenção: os valores apresentados são de referência e podem variar conforme distância, horário, condições da entrega e acordo entre as partes.

## Público-alvo

- Motoboys autônomos;
- Associação de Motoboys Autônomos Independentes;
- Clientes que desejam consultar valores de entrega com transparência.

## Acesso local

Como o projeto está em HTML estático, você pode abrir o arquivo `index.html` diretamente no navegador.
